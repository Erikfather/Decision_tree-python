# Decision_tree-python
决策树分类（ID3，C4.5，CART）
三种算法的区别如下：
ID3算法以信息增益为准则来进行选择划分属性，选择信息增益最大的；
C4.5算法先从候选划分属性中找出信息增益高于平均水平的属性，再从中选择增益率最高的；
CART算法使用“基尼指数”来选择划分属性，选择基尼值最小的属性作为划分属性
